# Odin Recipes

This project is part of The Odin Project curriculum. It is a simple website that showcases a collection of recipes. The goal of this project is to practice basic HTML structure and linking between pages.

## Project Structure

The project consists of the following files and directories:

- `index.html`: The homepage of the website, which links to individual recipe pages.
- `recipes/`: A directory containing individual recipe pages.
  - `shakshuka.html`: A recipe for Shakshuka.
  - `vietnamese-coffee.html`: A recipe for Vietnamese Coffee.
  - `mug-brownie.html`: A recipe for Mug Brownie.

## Features

- A homepage with links to all recipes.
- Individual pages for each recipe, including ingredients and preparation steps.

## Skills Practiced

- Basic HTML structure
- Linking between pages
- Organizing files and directories
